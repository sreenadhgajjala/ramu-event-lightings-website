# Ramu Event Lightings & Power Solutions Website

## Overview

This project is a professional business website created for Ramu Event Lightings & Power Solutions. The website helps showcase event lighting, wedding mandap setup, house wiring, and electrical services to potential customers.

## Business Details

* Business Name: Ramu Event Lightings & Power Solutions
* Industry: Event Lighting & Electrical Services
* Location: Andhra Pradesh, India

## Services Offered

* Wedding Lighting
* Function Lighting
* Festival Lighting
* Wedding Mandap Setup
* Stage Decoration
* House Wiring
* Electrical Repairs
* Home Electrical Maintenance

## Features

* Modern Responsive Design
* Professional Service Showcase
* Gallery Section
* Contact Form
* WhatsApp Integration
* Mobile Friendly Layout

## Tools Used

* Framer AI
* ChatGPT
* GitHub

## Prompt Used

The website was generated using AI-powered prompts focused on:

* Homepage Content
* Service Descriptions
* Call-to-Action Sections
* Business Branding

## Project Objective

To create a professional online presence for a local business and demonstrate the use of AI and Prompt Engineering in website development.
